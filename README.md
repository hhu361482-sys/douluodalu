# 斗罗大陆文字版 RPG

这是一个单文件 HTML 文字 RPG 游戏，适合直接使用 GitHub Pages 发布。

## 发布方法

1. 打开你的 GitHub 仓库，例如 `douluodalu`。
2. 把本文件夹里的所有文件上传到仓库根目录：
   - `index.html`
   - `.nojekyll`
   - `README.md`
3. 进入仓库：Settings → Pages。
4. Source 选择：Deploy from a branch。
5. Branch 选择：main。
6. Folder 选择：/ root。
7. 点击 Save。
8. 等几分钟后，访问：

```text
https://你的GitHub用户名.github.io/douluodalu/
```

## 注意

- `index.html` 必须放在仓库根目录。
- 不要把 `index.html` 放进子文件夹里，否则 Pages 可能找不到主页。
- 游戏存档保存在浏览器本地 localStorage 中，换浏览器或换设备时不会自动同步。

## 彩蛋

在“新的觉醒”中输入指定角色名，会触发隐藏彩蛋。
